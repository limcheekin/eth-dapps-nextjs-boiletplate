MetaMask Web3js Smart Contract Integration

The code here is adapted from an excellent article Build Your First Solidity Dapp With Web3.js and MetaMask by Adnan Siddiqi with minor modifications for the Greeter smart contract, full credit should be given to him.

  • First, if you are running the sample dApp from Google Chrome, you need to start the server from the client directory using npm run dev due to the CORS origin error.
  • Start the truffle development environment with truffle develop, run compile and migrate after it is started.
  • Copy the contract address under Deploying 'Greeter'section, something like 0xd3bb084045105850C2C9E59d5e5Ae8Bf90D40829 and update the let contractAddress = '0x...' at line 101.
  • You're ready! Go to localhost:3000/static.html.
